# Git Status

Run the commands under `Commands` and summarize the current state of the git repository.

## Commands

- Current status: !`git status`
- Current diff (branch vs origin/main): !`git diff HEAD origin/main`
- Current branch: !`git branch --show-current`

## Files

@README.md

## Report

- Branch + whether working tree is clean
- What changed (high level)
- Any untracked files of note
